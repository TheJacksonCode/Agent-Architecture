# Writer — Prompt do generowania ikony

## Agent: Writer (Technical Writer / Redaktor)
**Rola**: Agent jakosci tresci. Zamienia surowy tekst w profesjonalny dokument. Poprawnosc jezykowa, spojnosc stylu, czytelnosc techniczna. README, CHANGELOG, API docs.
**Kolor**: #FCD34D — zloty/yellow-gold
**Model**: Claude Opus 4.6

## PROMPT

```
Create a premium cosmic icon for "The Writer" — a documentation craftsman in a multi-agent AI system.

CONCEPT: A cosmic quill pen — an elegant writing instrument that transforms chaos into clarity, floating in space and leaving a trail of golden light as it writes. This agent is the sound engineer of the team — raw tracks from developers sound rough, the Writer mixes them into a polished, professional album. Every word intentional, every sentence clear.

VISUAL DESCRIPTION:
- Central element: A stylized pen/pencil — a diagonal line from bottom-left to top-right (the pen body), with a triangular nib at the bottom-left tip. The classic edit/compose icon, but refined: the pen body has a slight taper, thicker at top, sharper at the nib. The angle is ~45 degrees — the natural writing angle.
- Edit marks: A thin horizontal line near the top-right of the pen, crossing it — the classic "edit" crossbar that turns a pen into an editing tool. Below it, another short parallel line — representing revision marks.
- Written content: Below and to the left of the pen nib, 2 short horizontal dashes — representing text that has just been written/edited. Clean, aligned, polished output.
- Ink trail: A very subtle small curved line extending from the pen nib — the trail of creation, golden words flowing onto the page.
- Polish sparkle: A tiny star/sparkle (4-pointed) near the top of the pen — representing the "polish" this agent adds. Raw text goes in, professional documentation comes out.

STYLE:
- Galactic aesthetic — celestial calligraphy instrument
- Golden palette: primary #FCD34D, deep #F59E0B, light #FDE68A, accent #FEF3C7
- ELEGANT, refined, PRECISE — this icon should feel literary and polished
- Diagonal composition — dynamic angle suggesting active creation
- Warm golden glow — words are precious, documentation is gold

MOOD: Literary precision, content excellence — "README.md polished. CHANGELOG updated. API docs: 100% coverage."

TECHNICAL: SVG viewBox="0 0 24 24", stroke-width 1.5-2, currentColor, transparent bg.
```
